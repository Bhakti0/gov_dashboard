# gov_dashboard

1. Project Overview

This project is a simple AI Governance Prototype created to demonstrate how AI systems can be monitored and controlled safely. It includes:

1. AI Monitoring
2. Risk Assessment
3. Compliance & Ethical Rule Checking
4. Governance Decision (Approve / Reject)
5. Streamlit Dashboard for easy demo

This system helps show how AI decisions can be made safer and more responsible, which is an important concept in modern AI development.

2. What This Project Does

When a user enters a prompt (input):
1. The system monitors the input
2. It checks for sensitive content
3. It calculates a risk score
4. It checks compliance rules

Then gives a final governance decision

1. APPROVE — Safe
2. REJECT — Unsafe
3. ESCALATE — Needs human review
